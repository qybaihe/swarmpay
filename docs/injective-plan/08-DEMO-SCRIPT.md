## 录制要点
- **先 mock 录全流程**：保证一次过、不黑屏、节点都激活。
- **testnet 补镜头**：单独录一段真实 txHash 在浏览器里的画面，作为可信度亮点插入 2:35。
- **分润流向图**是视觉记忆点，务必有动画。
- 字幕/配音突出三个关键词：**蜂群协作 · 链上分润 · 越用越聪明**。

- 录屏：macOS 自带 QuickTime / OBS
- 编辑：剪映 / iMovie（加字幕、片头片尾）
- 导出：1080p，MP4
- 上传：YouTube（设未公开）或 Google Drive，贴链接进报名表单
